How to Fine-Tune LLMs Locally with LoRA/QLoRA (2026 Guide)

Piyush Sahoo • March 5, 2026 • ~18 min read

Running and fine-tuning large language models no longer requires expensive cloud GPUs. In this guide you'll learn how to fine-tune models like Llama 3.1 8B or Mistral Nemo on consumer hardware using efficient techniques (LoRA + 4-bit quantization).

Prerequisites

  • Python 3.10+
  • GPU with ≥8 GB VRAM (RTX 3060 / 4060 / 4070 recommended)
  • At least 32 GB system RAM for comfort

Step 1: Set Up the Environment

# Recommended: use uv (fastest) or conda
uv venv --python 3.11
source .venv/bin/activate

uv pip install torch==2.4.0 --index-url https://download.pytorch.org/whl/cu121
uv pip install transformers datasets peft accelerate bitsandbytes trl
Important: Use the correct torch CUDA version for your GPU. Check with nvidia-smi.
